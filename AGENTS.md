## CONVENTIONS

- **File naming**: lowercase, descriptive
- **Main sections**: `chapters/` directory
- **Bibliography**: `citations/ref.bib`
- **References**: Use `\cref{}` (cleveref) not `\ref{}`
- **Quotes**: Use `\enquote{}` (csquotes) not ````''

## ANTI-PATTERNS (THIS PROJECT)

- **Markdown残留** → Remove `##`, `**`, `[@...]` in .tex
- **Hardcoded references** → Use BibTeX labels
- **Missing compilation** → Always compile to verify
